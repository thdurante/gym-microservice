## Setup
Para configurar o projeto, siga os passos abaixo:
```
Na pasta raiz do projeto:
$ bundle intall
$ rails db:create
$ rails db:migrate
```
Versões: `ruby: 2.4.2` e `rails: 5.1.4`

## Run
Para rodar o projeto, siga os passos abaixo:
```
Na pasta raiz do projeto:
$ rails s
```
